Паттерн Visitor - поведенческий шаблон проектирования,
описывающий операцию, которая выполняется над определенными объектами коллекции.
В зависимости от типа элемента, может быть использованы разные посетители с разными алгоритмами работы.
Данный паттерн можно использовать совместно с паттерном Итератор.

Классы:
Visitor - интерфейс посетителя.
ConcreteViitor - конкретная реализация посетителя.
Element - интерфейс элемента.
ElementA - конкретный элемент номер один.
ElementB - конкретный элемент номер два.

Применение:
Применяется для обхода элементов коллекции по определеным алгоритмам.
