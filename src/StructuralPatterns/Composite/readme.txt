Паттерн Компоновщик - предназначен для формирование иерархического дерево из различных классов.

Классы:
Component - общий интерфейс, который должен реализовываться каждым компонентом дерева.
Leaf - класс простого элемента, который не может иметь дочерних узлов
Composite - класс сложного элемента, который может хранить ссылки на дочерние элементы.

Применение:
Паттерн предоставляет механизм создания структурированного дерева, состоящего из классов различных типов.

Абстракции:
Компоновщик, дерево.
