Паттерн Adapter - структурный шаблон проектирования,
используемый для применения класса без нужного интерфейса, в месте где этот интерфейс требуется.

Классы:
SourceInterface - исходный интерфейс. Объект с таким интерфейсом необохдимо использовать.
Adaptee - класс, который нужно адаптировать под интерфейс SourceInterface.
Adapter - класс, который адаптирует Adaptee под интерфейс SourceInterface.

Применение:
Используется когда нужно адаптировать класс под определенный интерфейс.

Все названия:
Адаптер
