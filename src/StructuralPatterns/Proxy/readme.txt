Паттерн Proxy - структурный шаблон проектирования,
основная задача данного паттерна в том, чтобы отложить инициализацию ресурсов до непосредственного использования объекта.

Классы:
Subject - интерфейс, который описывает исходный объект.
ConcreteSubject - конретный класс, реализующий Subject интерфейс.
Proxy - класс-заместитель, который наследует Subject и замещает собою ConcreteSubject.

Применение:
Применяется при необходимости отложенной инициализации конкретного класса.
Еще одно применение это анализ исходного объекта. Например при вызове определенных методов объекта,
информация о их параметрах записывается в базу данных.

Абстракции:
Заместитель, Суррогат, Оболочка
