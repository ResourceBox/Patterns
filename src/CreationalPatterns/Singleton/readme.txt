Паттерн Singleton - порождающий шаблон проектирования,
основная особенность данного паттерна в том, что он не позволяет создавать больше одного экземпляра объекта.

Классы:
Singleton - объект, который может иметь только один экземпляр.

Применение:
Применяется при необходимости недопустить создания второго экземпляра класса. (Например второго класса Calendar)

Абстракции:
Одиночка
