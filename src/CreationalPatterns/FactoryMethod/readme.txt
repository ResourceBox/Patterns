Паттерн FactoryMethod - порождающий шаблон проектирования,
основная особенность которого создавать объект с определенным интерфейсом в зависимости от входного параметра.

Классы:
Creator - интерфейс создателя, который порождает новые объекты.
ConcreteCreator - конкретный создатель.
Product - интерфейс продукта.
FirstConcreteProduct и SecondConcreteProduct - конкретные продукты.

Применение:
Применяется при необходимости скрыть тип объекта, и обеспечить обращение по общему интерфейсу.
//fromRequest() - фабричный метод, который создает разнообразные объекты с интерфейсом Response 
// в зависимости от входного параметра.
//Response response = Response.fromRequest(request);

Абстракции:
Фабричный метод
