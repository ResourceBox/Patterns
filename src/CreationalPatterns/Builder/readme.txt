Паттерн Builder - порождающий шаблон проектирования,
задача которого создавать и настраивать сложные объекты.

Классы:
Builder - интерфейс строителя.
ConcreteBuilder - реализация строителя. Каждый строитель носит в себе индивидуальные настройки объекта.
Director - интерфейс класса, управляющего построением объекта и позволяющим менять строителей.
ConcreteDirector - реализация директора.
Product - класс продукта.

Применение:
Применяется при необходимости настройки одного и того же объекта по разному. Для этого используется
набор строителей и в зависимости от ситуации применяется нужный. За выбор строителя отвечает класс ConcreteDirector.
