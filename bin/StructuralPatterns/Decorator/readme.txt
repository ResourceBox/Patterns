Паттерн Декоратор - предназначен для динамического подключения дополнительного поведения к объекту, путем оборачивания объекта в дополнительные классы. 
Шаблон Декоратор предоставляет гибкую альтернативу практике создания подклассов с целью расширения функциональности.

Классы:
Component - общий интерфейс, наследуемый классами и декораторами.
FirstComponent - первый исходный компонент.
SecondComponent - второй исходный компонент.
Decorator - класс, который позволяет добавлять новые методы к исходным классом.
FirstDecorator - первый класс, отнаследованный от декоратора, позволяет добавлять функциональность к методам класса Decorator.
SecondDecorator - второй класс, отнаследованный от декоратора, позволяет добавлять функциональность к методам класса Decorator.

Способ решения
Декоратор предусматривает расширение функциональности объекта без определения подклассов.

Применение:
Используется для расширения функционала определеного набора классов, наследуемого от одного интерфейса.
Каждый декоратор добавляет функциональность с помощью "оборачивания" исходного класса.

Абстракции:
Оболочка, Слой.
